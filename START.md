# מערכת שיבוץ חדרים — הוראות הפעלה

## דרישות מקדימות
- Node.js גרסה 18 ומעלה

## התקנה ראשונה (פעם אחת)

פתח טרמינל בתיקיית `room-system` והרץ:

```bash
cd server && npm install && cd ../client && npm install && cd ..
```

## הפעלה

**אפשרות א — שני טרמינלים:**

טרמינל 1 (שרת):
```bash
cd server
npm run dev
```

טרמינל 2 (קליינט):
```bash
cd client
npm run dev
```

פתח דפדפן: http://localhost:5173

## מה צריך לעשות אחרי ההתקנה

1. **הגדרת עובדים** — לך לטאב "עובדים" והוסף את כל 50 העובדים עם תפקידיהם
2. **הגדרת חדרים** — ב"חדרים" תוכל לשנות שמות/סוגים של 24 החדרים
3. **כל עובד** נכנס למערכת ומגדיר את ימי/שעות הנוכחות שלו בטאב "הלוח שלי"
4. **הפעלת האלגוריתם** — לאחר שכל העובדים הגדירו לוח זמנים, לחץ "הפעל אלגוריתם שיבוץ"
5. **בדוק התנגשויות** — המערכת תציג אם יש עובדים שלא שובצו ותסביר למה

## מבנה תיקיות

```
room-system/
├── server/          ← Express + SQLite
│   ├── data/        ← קובץ ה-DB (נוצר אוטומטית)
│   ├── uploads/     ← קבצים שהועלו
│   └── routes/      ← API endpoints
└── client/          ← React + Tailwind
    └── src/
        ├── pages/admin/      ← דפי מנהל
        └── pages/employee/   ← דפי עובד
```
